# Gitfollow

Github following and follower control


## Installation
### Source Code
- Download  [Latest Source ](https://github.com/sepandhaghighi/gitfollow/archive/master.zip)

- `python3 setup.py install`

## Usage ##
- Enter your username & wait
- Find your data in log files
